## Step 로그

### Step 001 - 기본 문장 엔진: want to / need to / can

파일:
- `steps/step-001-basic-patterns.md`

상태:
- 수업 완료
- 숙제 제출 완료
- 첨삭 완료

핵심 성과:
- want to 패턴은 어느 정도 사용 가능.
- can 패턴 감 잡음.
- because 문장 사용 가능.

누적 약점:
- I, English 대문자 습관 필요.
- more words 어순 주의.
- for 30 minutes에서 for 누락.
- with / after / in 전치사 사용 흔들림.
- want to 뒤에 동사 누락: I want to my idea X.

### Step 002 - 전치사 기본: with / after / for / in

파일:
- `steps/step-002-prepositions.md`

상태:
- 수업 제공 완료
- 1차 숙제 제출 완료
- 1차 첨삭 완료
- Step 002-2 반복 숙제 제출 완료
- Step 002-2 첨삭 완료
- Step 002-3 최종 확인 진행 중

핵심 목표:
- with = ~와
- after = ~후에
- for = 시간 길이
- in = 영어로/장소/기간 안에

1차 숙제 평가:
- with / after / in은 일부 문장에서 잘 사용함.
- `I need to read English documents with my team.`은 좋음.
- for 시간 길이, want to 뒤 동사, listen to, at work가 아직 흔들림.
- Step 002는 아직 완료하지 않고 Step 002-2 반복 훈련으로 진행.

Step 002-2 평가:
- `in English`, `listen to`, `at work`, `ten words every day`가 개선됨.
- 2, 5, 6, 7, 8번은 거의 맞거나 정확함.
- 아직 `for one hour`, `want to` vs `need to`, this/these 구분 확인 필요.
- Step 002-3 짧은 최종 확인 후 통과 가능성이 높음.

## 반복 오류 누적 목록

1. 대문자
   - i X → I O
   - english X → English O

2. 동사 누락
   - I can short sentence X
   - I can write short sentences O
   - I want to my idea X
   - I want to express my ideas O

3. 전치사
   - by foreigners X → with foreigners O
   - by work out X → after work O
   - 30 minutes X → for 30 minutes O
   - one hour X → for one hour O
   - that english X → in English O
   - in work X → at work O
   - listen English news X → listen to English news O

4. 어순
   - words more X → more words O
   - words for ten X → ten words O

5. 단수/복수
   - this documents X → this document O
