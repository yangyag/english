# 현민 영어 학습 프로필

Last updated: 2026-05-15 17:05:11 KST (+0900)

## 현재 영어 레벨

대략 CEFR 기준: A1 후반 ~ A2 초반

## 특징

- 프로그래머라서 영어 단어와 기술 문서를 많이 본 경험은 있음.
- 읽기 경험은 어느 정도 있으나, 직접 문장을 만드는 힘이 약함.
- 쓰기/말하기는 A1~A2 초반 수준.
- 문법 기초가 흔들림.
- 개발/업무 단어는 익숙하지만 일반 문장 어휘는 부족.

## 목표

- 영어 문서를 덜 힘들게 읽기.
- 기본 회화를 입에서 나오게 만들기.
- 개발 문서, 에러 메시지, API docs, infra docs를 더 편하게 읽기.
- 외국인과 기본적인 대화를 자연스럽게 하기.

## 주요 약점

### 1. be동사와 일반동사 혼동

틀린 예:
- I am go to work yesterday.

맞는 예:
- I went to work yesterday.
- I am tired.
- I go to work.

### 2. want / need / can 패턴

- I want to + 동사원형
- I need to + 동사원형
- I can + 동사원형

주의:
- want study X
- want to study O
- need study X
- need to study O
- can to study X
- can study O

### 3. 3인칭 단수

틀린 예:
- He don’t like coffee.
- He doesn’t likes coffee.

맞는 예:
- He doesn’t like coffee.

### 4. 명사/동사 구분

틀린 예:
- I want to English very well.

맞는 예:
- I want to speak English well.
- I want to be good at English.

### 5. 전치사와 자연스러운 표현

틀린 예:
- I usually use work.
- I want to speaking English foreigner.

맞는 예:
- I usually use English at work.
- I want to speak English with foreigners.
