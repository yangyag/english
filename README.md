# 현민 영어 학습 인덱스

Last updated: 2026-05-15 17:54:32 KST (+0900)

이제 영어 학습은 Day가 아니라 Step 기준으로 진행한다.
하루에 시간이 많으면 Step 2개, 3개를 진행해도 된다.
시간이 적으면 한 Step을 며칠에 나눠서 해도 된다.

## 현재 상태

- 현재 레벨: A1 후반 ~ A2 초반
- 현재 진행: Step 002-3 최종 확인 진행 중
- 현재 숙제: `steps/step-002-prepositions.md`의 Step 002-3 최종 확인 숙제 확인
- 전체 진도 기록: `progress.md` 확인

## 파일 구조

- `README.md`
  - 전체 인덱스. 다음에 어디서 이어갈지 확인하는 파일.

- `profile.md`
  - 현재 영어 능력, 약점, 목표.

- `plan.md`
  - 3개월 학습 방향, 30분/60분/120분 루틴, Step 운영 규칙.

- `teaching-method.md`
  - 수업 방식과 의도. 강의/출력/첨삭/실전 적용을 어떻게 섞을지 기록.

- `progress.md`
  - Step별 완료/진행 상태, 숙제 제출/첨삭 로그.

- `steps/`
  - 각 Step별 수업 내용과 숙제.
  - 예: `step-001-basic-patterns.md`, `step-002-prepositions.md`

- `archive/`
  - 예전 단일 파일 구조 백업.
